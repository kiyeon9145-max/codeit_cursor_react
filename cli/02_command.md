# 파일 시스템

- 나 지금 어디에 있지?
  - pwd

- 지금 여기에 뭐가 있지?
  - ls
  - ls -l: 자세히 볼 수 있다.
  - ls -a: 숨김 파일 또는 폴더 보기.

- 나 다른 폴더로 이동할래 
  - cd [이동하고싶은폴더경로]
    - .: 현재 디렉토리
    - ..: 이전 디렉토리
    - /: 루트 디렉토리
    - ~: 홈 디렉토리(사용자 디렉토리)

- 폴더를 만들어보자
  - mkdir [만들고싶은폴더이름]

- 파일을 만들어보자
  - touch [만들고싶은파일이름]

- 파일 이름을 변경해보자.
  - mv [현재이름] [바꿀이름]

- 파일을 이동해보자.
  - mv -i [옮길파일] [옮길경로]

- 파일을 복사 붙여넣기 해보자.
  - cp [복사할파일] [복사할경로]

- 폴더를 복사 붙여넣기 해보자
  - cp -r [복사할폴더] [복사할새로운폴더이름]

- 삭제해보자.
  - rm [파일]
  - rm -r [폴더]: 폴더 내부 전부 삭제
  - rm -f [폴더]: 폴더 강제 삭제
