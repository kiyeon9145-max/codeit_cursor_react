# 파일 시스템

- 나 지금 어디에 있지?
  - pwd

- 지금 여기에 뭐가 있지?
  - ls
  - ls -l: 자세히 볼 수 있다.
  - ls -a: 숨김 파일 또는 폴더 보기.

- 나 다른 폴더로 이동할래 
  - cd [이동하고싶은폴더경로]
    - .: 현재 디렉토리
    - ..: 이전 디렉토리
    - /: 루트 디렉토리
    - ~: 홈 디렉토리(사용자 디렉토리)

- 폴더를 만들어보자
  - mkdir [만들고싶은폴더이름]

- 파일을 만들어보자
  - touch [만들고싶은파일이름]
