# 운영체제

## 운영체제란?

- 사용자 프로그램이 컴퓨터의 하드웨어에 접근하는 것을 추상화하는 프로그램
  - 사용자 프로그램 <-> 운영체제 <-> 하드웨어
  - js, java, python... <-> c언어로 만들어진 사용자 프로그램 <-> 바이너리 프로그램(실행파일)
    <-> 운영체제 <-> 하드웨어
  - 터미널: 사용자가 명령어를 통해서 운영체제랑 소통할 수 있도록 도와주는 프로그램
  - shell: 터미널에 입력된 명령어를 운영체제가 알아들을 수 있도록 해석해주는 소프트웨어
   - 윈도우: power shell
   - mac: zsh
   - 우분투: bash
